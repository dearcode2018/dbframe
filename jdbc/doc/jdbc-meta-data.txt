

表元数据-结果集列名称
TABLE_CAT -- catalog
TABLE_SCHEM -- schema
TABLE_NAME -- table name
TABLE_TYPE - table type
REMARKS -- comment

DATA_TYPE
TYPE_NAME
COLUMN_SIZE

DECIMAL_DIGITS
NUM_PREC_RADIX
NULLABLE

COLUMN_DEF
SQL_DATA_TYPE
SQL_DATETIME_SUB
CHAR_OCTET_LENGTH
ORDINAL_POSITION
IS_NULLABLE

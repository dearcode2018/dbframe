【项目信息】
项目名称 : database-split
中文名称 : 数据库 拆分
项目版本 : version 1.0

主要技术 : 数据库公共特性、标准sql语句、数据库公共参数、通用场景

项目描述 : 
数据库拆分，分库，分表.
多个库、多个表
水平拆分: 一个库、表结构复制为多个，根据数据特定进行划分
数据库水平拆分、数据表水平拆分


垂直拆分: 根据业务领域进行划分
数据库垂直拆分、数据表垂直拆分


项目备注 : 

【项目规划】

spring 多数据源配置 (不适用与分库分表 dao公用一套，它是每个dao对应相应的数据源)
http://blog.csdn.net/wangpeng047/article/details/8866239
spring 多数据源配置适用于不共用dao的情况，实体可以共用，一个dao对应一个数据源，事务在dao层或service层
进行单独指定
